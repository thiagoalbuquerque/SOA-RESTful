IMPLEMENTAÇÃO DE SERVIÇOS RESTFUL

IMPORTANTE: 
- DEVEM SER CADASTRADOS DADOS REAIS NA BASE DADOS.
- DEVE SER ENTREGUE DOCUMENTAÇÃO DOS SERVIÇOS EM ARQUIVO WORD COMO DESCRITO NOS ITENS DO TRABALHO.

########################################################################################################

1 - De acordo com Josuttis (2007) defina serviço básico, serviço básico de dados, serviço básico de lógica, serviço
composto e serviço de processo.
Observação: A resposta deve ser apresentada no arquivo de documentação.

#####################################################################################################################

2 - Baseado na implementação dos serviços do trabalho 3, implemente serviços básico de acesso a dados empregando os
conceitos de serviços RESTful. Realize os seguintes passos:
- Construir um banco de dados que seja capaz de armazenar informações de autores e publicações. Publicações devem
armazenar a sua classificação Qualis.
- Implementação de classe DAO para executar operações CRUD (Create, Retrieve, Update e Delete) sobre a tabela Publicacao
e Autor
- Implementação de métodos do serviço para encapsular o acesso ao banco de dados.
- Aplicação cliente para invocar as operações CRUD expostas pelo serviço.

Observações:
- São apresentados exemplos de publicações no final deste documento que podem ser usados para a construção do banco de
dados.
- O script de criação do banco de dados e carga de dados reais devem ser apresentados no arquivo de documentação.
- Os endpoints para os serviços devem ser apresentados no arquivo de documentação bem como a estrutura dos dados de
entrada e de saída, além de um exemplo de chamada do serviço.
- Os endpoints do serviço devem considerar os princípios RESTful.

6 - Implemente serviço composto que calcule a pontuação de um autor, ou seja, ele deve invocar serviço que obtém as
publicações dos autores e, em seguida, invocar serviço que calcula a pontuação de um autor.

Ao realizar este trabalho o aluno terá:
- Praticadoos conceitos RESTful
- Configurado o ambiente de desenvolvimento
- Implementado serviços web RESTful de dados e de lógica.
- Implementado cliente para o serviço.
- Praticado a invocação remota de métodos entre cliente e consumidor, simulado no laptop do aluno.
- Exposto o acesso a um banco de dados através de uma camada de serviços de dados.
- Entendido na prática conceitos importantes em uma arquitetura orientada a serviços.
